# Social Media App

##  About the Project

This is a simple social media application where users can create an account, log in, and share photos. Users can interact with each other by liking and commenting on photos. Only logged-in users can access profiles and use the app’s features.

This project was built to practice user authentication and basic social media functionality.

---

##  Features

- User sign up and login
- Create and update a user profile
- Upload and post photos
- Edit or delete your own photos
- Like photos
- Comment on your own photos and other users’ photos
- Update profile picture
- View other users’ profiles (authentication required)

---

##  Authentication

- Users must be logged in to view profiles and posts
- Users can only edit or delete their own posts

---

## Getting Started

1. Create an account or log in.
2. Upload a photo.
3. Like and comment on posts.
4. Update your profile picture and manage your posts.

---

##  Purpose

This project is meant for learning and practicing:
- User authentication
- CRUD operations (Create, Read, Update, Delete)
- Basic social media interactions
